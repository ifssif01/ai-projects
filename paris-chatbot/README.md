# Paris Chatbot

A conversational travel guide for Paris, France, powered by OpenAI's GPT-4o-mini.

## What it does

Asks a series of questions about Parisian landmarks and attractions, maintaining conversation history so each response is context-aware.

## Requirements

- Python 3.x
- `openai` Python package

```bash
pip install openai
```

- An OpenAI API key set as an environment variable:

```bash
export OPENAI_API_KEY=your_key_here
```

## Usage

```bash
python paris_chatbot.py
```
